# Store Model (Python)

This project is a simple Python model of a store that sells books, music albums, and software licenses.  
Each product has basic properties, and a shopping cart can be used to add or remove items and calculate the total price and weight.

## Files

- `product.py` – Base product class  
- `book.py` – Book product  
- `music_album.py` – Music album product  
- `software_license.py` – Software license product  
- `shopping_cart.py` – Shopping cart implementation  
- `recommendation.py` – (Optional) function to analyse product order  
- `main_test.py` – Example usage

## How to Run

```bash
python main_test.py
